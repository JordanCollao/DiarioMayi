Configuración inicial

casi listo

Faltan dos cosas para conectar la base de datos.

1. Crear el proyecto en Supabase

  1. Entra a supabase.com y crea un proyecto (free tier).
  2. Cuando esté listo, ve a Settings → API.
  3. Copia Project URL y la key anon / public.

2. Pegar las credenciales en el HTML

Abre este archivo y reemplaza estas líneas (alrededor de la línea con SUPABASE_URL):

const SUPABASE_URL = 'https://tu-proyecto.supabase.co';
const SUPABASE_ANON_KEY = 'eyJhbGciOi...tu-anon-key';

3. Crear la tabla en Supabase

En el dashboard de Supabase, abre SQL Editor, pega esto y dale Run:

create table cafes (
  id uuid primary key default gen_random_uuid(),
  user_id uuid references auth.users on delete cascade
    not null default auth.uid(),
  name text not null,
  rating numeric(3,1) not null check (rating >= 0 and rating <= 10),
  fav smallint not null default 0 check (fav in (0,1,2)),
  notes text default '',
  photo text,
  date date,
  created_at timestamptz not null default now()
);

create index cafes_user_created on cafes(user_id, created_at desc);

alter table cafes enable row level security;

create policy "own cafes" on cafes
  for all using (auth.uid() = user_id)
  with check (auth.uid() = user_id);

4. Recarga esta página

Después de guardar los cambios en el HTML, recarga y aparecerá la pantalla de inicio de sesión.
